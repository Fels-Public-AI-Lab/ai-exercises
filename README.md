# AI Exercises

Interactive, browser-based exercises for the **Fels Institute of Government Public AI Lab** AI Literacy program at Penn LPS. Each exercise is a self-contained HTML file — no installation or build step required.

## Exercises

| # | Exercise | Folder |
|---|----------|--------|
| 1 | **The Prompt Lab** — Experiment with how prompt wording, context, and structure shape AI output | `prompt-engineering-lab/` |
| 2 | **The Hallucination Hunt** — Identify and analyze AI-generated content that is plausible but factually wrong | `hallucination-hunt/` |
| 3 | **The Framing Audit** — Examine how framing and perspective influence AI responses | `framing-audit/` |
| 4 | **AI as Research Partner** — Practice using AI effectively as a research and analysis tool | `ai-research-partner/` |

## Usage

Open any `index.html` file directly in a browser — no server needed.

## About

Developed by the [Fels Public AI Lab](https://github.com/Fels-Public-AI-Lab) at the University of Pennsylvania's College of Liberal and Professional Studies (Penn LPS).
